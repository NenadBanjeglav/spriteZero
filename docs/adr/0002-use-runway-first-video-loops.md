# Use Runway-First Video Loops

The campaign should feel cinematic enough to make visitors want Sprite Zero in Serbia immediately, so v1 will use short Runway-generated video loops for the most important story moments instead of relying only on static images. Still frames and GSAP overlays remain part of the workflow for speed, responsiveness, and performance. This trades extra asset cost and iteration time for stronger emotional impact in the hero and key story sections.
