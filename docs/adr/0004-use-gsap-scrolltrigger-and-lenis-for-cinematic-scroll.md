# Use GSAP ScrollTrigger And Lenis For Cinematic Scroll

The Expanded Cinematic Story Arc needs pinned scenes, scrubbed text beats, held transitions, and smooth scroll coordination that CSS scroll timelines cannot reliably provide across the campaign. GSAP ScrollTrigger will own cinematic scroll animation and Lenis will own smooth scrolling, while normal React UI continues to own voting, forms, sharing, accessibility states, and data rendering so demand interactions are not blocked by the film layer.
